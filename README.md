- `screenshots/` — папка со скриншотами работающего приложения (каталог, фильтры, корзина).


## Ответы на контрольные вопросы
1. **В чём разница между `setTimeout()` и `setInterval()`?**
   `setTimeout()` выполняет код один раз после указанной задержки. `setInterval()` выполняет код многократно через равные промежутки времени.

2. **Что произойдёт, если не вызвать `clearInterval()`?**
   Таймер продолжит работать в фоновом режиме, даже если он больше не нужен. Это приводит к утечкам памяти и лишней нагрузке на процессор.

3. **Что такое BOM?**
   Browser Object Model — это объекты, предоставляемые браузером для управления окном и получения информации о среде (window, location, navigator, screen, history).

4. **Как узнать, с мобильного ли устройства зашёл пользователь?**
   С помощью свойства `navigator.userAgent` и проверки его через регулярное выражение на наличие ключевых слов (Android, iPhone, iPad).

5. **Почему `alert()`, `confirm()` и `prompt()` считаются устаревшими?**
   Они блокируют выполнение основного потока (страницу нельзя прокрутить, нельзя кликнуть), их невозможно стилизовать, и они выглядят по-разному в разных браузерах.

6. **Зачем нужен метод `padStart()`?**
   Для дополнения строки символом до нужной длины. Это необходимо, чтобы время 00:01:05 выглядело корректно (с ведущими нулями).

7. **Что такое `location.href`?**
   Это свойство, которое хранит полный URL-адрес текущей страницы. Присвоив ему новое значение, можно перенаправить пользователя на другой сайт.
